# Fytò Inc. | Digital Engineering Portfolio

### Cultivating Modern Digital Solutions
A full-stack personal brand portfolio showcasing engineering growth and technical versatility.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3 (Silky Gradient Aesthetic), Vanilla JavaScript
- **Backend:** Python, FastAPI, Pydantic, SMTP Integration
- **OS:** Developed on Zorin OS (Linux)

## 🚀 Key Features
- **Project Greenhouse:** Dynamic project loading via JSON.
- **Signal Connection:** Asynchronous contact form with automated email notifications.
- **Responsivity:** Fully optimized for mobile and desktop "silky" experiences.
